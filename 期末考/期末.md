# 問答題:
```
1. 說明檔案處理系統與資料庫 系統 這兩種方法的差異 [要說明優缺點]
答:(1)檔案處理系統:具有重複的資料，以及資料被隔離。資料重複的情形可能提高錯誤的發生機率，也會導致如儲存空間和時間等資源的浪費。
   (2)資料庫：降低資料重複性、改進資料完整性、資料共享、容易存取，以及縮短開發時間。
```
```
2. 說明大數據 的 3V 特性與 5V特性
(1)Volume(數量):
指的是個人和機構產生的資料量，非結構數據的超大規模和增長，
總數據的80~90% ，比結構化數據增長快10倍到50倍，是傳統數據倉庫的10倍到50倍。
(2)Value(價值):
大量的不相關信息，對未來趨勢與模式的可預測分析，深度複雜分析（機器學習，人工智慧vs傳統商務智能）
(3)Variety(多樣性)
是指資料能呈現或儲存的各種格式，以便於人們或電腦程式運用。
(4)Velocity(速度):
是指處理資料的速率
(5)Veracity(真實性):
大數據中的內容是與真實世界中的發生息息相關的，研究大數據就是從龐大的網絡數據中提取出能夠解釋和預測現實事件的過程。
```
```
3. 說明系統開發生命週期（system development life cycle）有哪些階段 ?每階段要做甚麼事?
(1)規劃階段（planning phase）
•	審查和核准專案要求
•	決定專案要求的優先順序
•	分配資源
•	成立專案開發團隊
(2)分析階段（analysis phase）
(一)進行初步調查
•	主要目的是確認問題或改進方案的真正本質，並決定是否值得繼續進行
•	訪談提出專案要求的使用者
(二)執行細部分析
•	研究目前系統是如何運作的
•	找出使用者的期望、需要和需求
•	建議解決方
(3)設計階段（design phase）
•	取得需要的硬體及軟體
•	制訂新系統或即將修改的資訊系統的所有細節
(4)實作階段（implementation phase）
•	開發程式和 App
•	使用者教育訓練
•	轉換到新系統
(5)技術支援及安全防護階段（support and security phase）
•	進行維護活動
•	監控系統效能
•	評估系統安全性
```
```
4. 請舉 程式語言 例說明編譯器 （compiler）與直譯器（interpiler）的差別:
•	編譯器（compiler）是一個分開獨立的程式，用途是將整個原始碼程式一次轉換成能被執行的機器語言
Ex：c語言、c++
•	直譯器（interpreter）是每次讀取一條程式敘述，將其轉換成一或多條機器語言指令，然後執行那些機器語言指令
Ex：Python、Ruby
```
